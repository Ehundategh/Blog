# 抽象代数

## 群

给定一个集合 $G$ 和定义在集合中的元素的运算 $G\circ G\to G$，我们称 $(G,\circ)$ 为一个群，当且仅当这个运算是封闭的、结合的，且有幺元和逆元。

例如，$(\mathbb{N},+)$ 不是一个群。虽然其封闭，且满足结合律，还有幺元 $0$，但是它没有逆元。对于一个自然数 $a>0$，若要 $a+b=0$，则 $b=-a<0$，不在 $\mathbb{N}$ 中。

而 $(\mathbb{Z},+)$ 就是一个群。

## 半群

削弱限制，我们定义半群 $(G,\circ)$，只需要 $\circ$ 是封闭且结合的即可。

加强限制，我们定义 Abel 群，即群 $(G,\circ)$ 的运算 $\circ$ 符合交换律的特殊情形。

交换性还可以约束在半群上。于是我们同样定义 Abel 半群，即半群 $(G,\circ)$ 的运算 $\circ$ 符合交换律的特殊情形。

## 幺半群与子群

对于有幺元 $e$ 的半群 $(G,\circ)$，称其为幺半群。

对于 $G'\subseteq G$，若成立 $(G',\circ)$ 是一个群，则称 $(G',\circ)$ 是 $(G,\circ)$ 的子群。

事实上，在算法中我们更关心的是封闭性。若一个信息集合在合并或者打标记后会跑出原集合，那么原本的数据结构维护方式就不能直接套用。

## 基本性质

群中元素有一些统一的性质。若 $e$ 是幺元，$a^{-1}$ 是 $a$ 的逆元，则有：

$$a^{-1}\circ a=a\circ a^{-1}=e$$

以及：

$$e\circ a=a\circ e=a$$

也就是说，幺元和逆元在对应位置上有天然的对称性。这里写出这些性质，是为了说明我们在后续使用代数结构时，并不需要每次重新证明这些基础结论。

关于群还可以拓展出环、域以及相应理论。通过群论中的 Burnside 引理还可以得到 Pólya 计数原理，用以解决一类计数问题。这里不再赘述。
