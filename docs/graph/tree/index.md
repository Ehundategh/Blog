# 树上问题

树是图论中最基础的一类结构。由于树上任意两点之间只有一条简单路径，很多图上的问题在树上会表现出更强的性质。

## 树基础

我们先给出无根树最基础的定义：含有 $n$ 个节点的最小连通结构。根据这个定义，很显然，若要求结构最小，那么一定只能含有 $n-1$ 条边。

根据树的定义，可以知道以下说法等价：

- 树是含有 $n$ 个节点和 $n-1$ 条边的连通图。
- 树是一个无向无环的连通图。
- 树中任意一条边均为桥。
- 树中任意两个点均仅存在一条简单路径。

## 有根树

有根树就是定了一个节点为根的无根树。在有根树中，有如下定义：

- 叶子节点：不为根且度数不超过 $1$ 的节点。
- 父亲：对于节点 $u$，若 $v$ 与 $u$ 邻接，且 $v$ 在 $u$ 到根节点路径上，称 $v$ 是 $u$ 的父亲，记作 $v=\operatorname{fa}(u)$。
- 祖先与后代：对于节点 $u$，若 $v$ 在 $u$ 到根节点的路径上，则称 $v$ 为 $u$ 的祖先，记作 $v\in\operatorname{anc}(u)$，同时称 $u$ 是 $v$ 的后代。
- 子树：对于节点 $u$，其子树是去掉边 $(u,\operatorname{fa}(u))$ 之后 $u$ 所在的树，记作 $\operatorname{subtree}(u)$。
- 管辖：所有满足 $\operatorname{fa}(u)=v$ 的 $u$ 构成了 $v$ 的管辖节点集合，记作 $u\in\operatorname{son}(v)$。

## 常见形态

- 森林：可以被划分为若干棵树的图。
- 链：任一节点的度数不超过 $2$ 的树。
- 菊花：只有一个节点度数不为 $1$ 的树。
- 二叉树：每个节点至多有 $2$ 个管辖节点的树。
- 多叉树：不限制节点管辖节点个数的树。

## 左儿子右兄弟表示法

一般而言，我们处理二叉树上的问题比多叉树更简单，所以可以考虑一种把多叉树转化成二叉树的方法。

对于任意节点 $u$，对 $\operatorname{son}(u)$ 任意排序，令排序为 $\{v_1,v_2,\cdots,v_k\}$。用 $v_1'$ 作为新树中 $u'$ 的左儿子，同时令 $v_i'$ 的右儿子为 $v_{i+1}'$。可以验证，新得到的一棵树是二叉树。
